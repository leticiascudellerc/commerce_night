# Beyond the Waves — CUS Commerce Night 2026

A pre-event landing page for the 4th Annual Commerce Night, hosted by the Commerce Undergraduate Society (CUS) at the University of British Columbia.

## About the Project

This landing page was built as part of my application for the VP Technology position at Commerce Night. It demonstrates my ability to design and develop a full-stack web experience from scratch, under a real deadline, for a real event.

The page was built in a few hours, entirely in vanilla HTML, CSS, and JavaScript.

## Features

- Underwater animated hero section with floating bubbles
- Smooth scroll navigation with gold glow hover effects
- Animated stat counters triggered on scroll
- Full evening itinerary with a glowing scroll-animated timeline
- Custom awards slider built without any libraries
- Reminders section with gold-bordered cards
- Tickets CTA section
- Fully responsive design

## Tech Stack

- HTML5
- CSS3 (custom animations, CSS variables, Grid, Flexbox)
- Vanilla JavaScript (IntersectionObserver, custom slider, counter animation)

## Design

Inspired by the official Commerce Night 2026 "Beyond the Waves" branding — deep navy, gold accents, and an ocean theme throughout.
